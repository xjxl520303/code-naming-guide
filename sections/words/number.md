## `通用`

```
number: 数字

integer: 整数

natural number: 自然数
rational number: 有理数
irrational number: 无理数
cardinal number: 基数
complex number: 复数
ordinal number: 序数
positive integer: 正整数
non-negative integer: 非负数
even: 偶数
odd: 奇数
infinity: 无穷
unit: 单位
```

## 计数单位

```
pixel: 像素(px)
percent: 百分比(%)
points: 品脱(pt)
inch: 英寸(in)
ems: 相对根元素来定义(em)
meter: 米(m)
centimeter: 厘米(cm)
decimetre: 分米(dm)
meter: 分米
cubic: 立方
square: 平方

<!-- 中文数字 -->
one: 个
ten: 十
hundred: 百
thousand: 千
ten thousand: 万
billion: 亿
trillion: 兆

<!-- 存储容量 -->
byte: 字节 B
// 二进制表示
kibibyte:1024 KiB
mebibyte: 1024^2 MiB
gibibyte: 1024^3 GiB
tebibyte: 1024^4 TiB
pebibyte: 1024^5 PiB
exbibyte: 1024^6 EiB
zebibyte: 1024^7 ZiB
yobibyte: 1024^8 YiB
// 十进制表示
kibobyte: 1000 kB
megabyte: 1000^2 MB
gigabyte: 1000^3 GB
terabyte: 1000^4 TB
petabyte: 1000^5 PB
exabyte: 1000^6 EB
zettabyte: 1000^7 ZB
yottabyte: 1000^8 YB
// 遗留问题
kibobyte: KB
megabyte: MB
gigabyte: GB
terabyte: TB
```
注意：`kB`、`KiB`和`KB` 3者的区别的。`KB`就是我们通常意义上理解和所使用的，而`KiB`更加专业，我们称之为**术语**，而`kB`则是历史问题，详见 [Files size units: "KiB" vs "KB" vs "kB"](https://ux.stackexchange.com/questions/13815/files-size-units-kib-vs-kb-vs-kb)
